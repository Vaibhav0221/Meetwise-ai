## 🎯 Problem Statement

Meetings often generate large amounts of unstructured data, leading to:

- ❌ Missed key decisions and action items  
- ⏱️ Time-consuming manual note-taking  
- 📉 Reduced productivity and clarity  

---

## 💡 Solution

MeetWise AI leverages a **multi-agent architecture** to automate:

- 📌 Meeting Summarization  
- 📊 Slide Content Generation  
- ✅ Action Item Extraction  
- 🧠 Insight Generation
